## Installation

![npm badge][npm-badge-url]

```bash
pnpm add -g http-server-md
```

### [Developing](#developing)

```bash
pnpm gen-readme // update README.md
pnpm docs // update DOCUMENTATION.md
pnpm test // lint & mocha
pnpm update-deps // bump all deps
```
